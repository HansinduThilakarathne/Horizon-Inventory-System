## 🆘 TROUBLESHOOTING

### Problem: "Port 5000/5173 already in use"

**Solution - Kill existing processes:**
```powershell
Get-Process node | Stop-Process -Force
```

Then restart the servers.

### Problem: "Can't reach http://localhost:5173"

**Solution - Check if Vite is running:**
- Frontend terminal should show "VITE v8.0.3 ready"
- If not, run: `npm run dev`

### Problem: "Registration fails with error"

**Solution - Check if backend is running:**
- Backend terminal should show "Backend Server running"
- If not, run: `npm start` in backend folder

### Problem: "npm: command not found"

**Solution - Install Node.js:**
1. Download from https://nodejs.org/
2. Install Node.js (includes npm)
3. Restart terminal
4. Try again
